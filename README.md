# AWS Cloud Security Lab

## Overview
This project demonstrates hands-on cloud security concepts using AWS Identity and Access Management (IAM), S3 bucket permissions, MFA, and CloudTrail logging.

## Objectives
- Implement role-based access control (RBAC)
- Enforce least privilege access
- Configure multi-factor authentication (MFA)
- Secure S3 bucket permissions
- Monitor user activity using CloudTrail

## Technologies Used
- AWS IAM
- Amazon S3
- AWS CloudTrail
- MFA (Multi-Factor Authentication)

## Security Features Implemented
- Created IAM users and groups
- Configured custom least privilege policies
- Enabled MFA for IAM users
- Blocked public access to S3 buckets
- Tested permission restrictions
- Enabled CloudTrail logging and monitoring

## Project Artifacts
- IAM users and MFA configuration
- Custom least privilege IAM policies
- S3 bucket security configuration
- Access control testing
- CloudTrail logging and event monitoring

## Key Takeaways
- Learned AWS IAM and access management fundamentals
- Applied least privilege security principles
- Configured cloud logging and monitoring with CloudTrail
- Gained hands-on experience with cloud security controls
